Privacy Policy – Secret Scanner

Secret Scanner is a Chrome extension designed to scan GitHub, GitLab, and VS Code web pages for exposed credentials such as API keys and tokens.

✅ Information Collection

Secret Scanner does not collect, store, or transmit any personal data.

The extension does not collect names, emails, passwords, browsing history, or any form of personally identifiable information.

All scanning is performed locally in the user’s browser.

✅ Website Content Access

The extension reads visible code and text on supported repository pages only for the purpose of detecting exposed credentials.

No website content is sent to external servers or third parties.

✅ Local Storage

Scan history and extension settings may be saved using Chrome’s local storage API.

This data is stored only on the user’s device and is never shared.

✅ Third Parties

Secret Scanner does not share, sell, or transfer any data to third parties.

✅ Remote Code

Secret Scanner does not load or execute any remote code.

✅ Contact

If you have any questions about this privacy policy, you may contact the developer through the Chrome Web Store listing.
